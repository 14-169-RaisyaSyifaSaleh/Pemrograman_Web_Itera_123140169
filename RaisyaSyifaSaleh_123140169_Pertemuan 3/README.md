Deskripsi Project

Furry Friends Explorer adalah aplikasi web interaktif yang memungkinkan pengguna untuk menjelajahi gambar-gambar lucu hewan peliharaan (anjing dan kucing) beserta fakta-fakta menarik tentang mereka. Aplikasi ini dibangun menggunakan ReactJS dengan Vite sebagai build tool dan mengintegrasikan beberapa API publik gratis.

Langkah Instalasi

1. Clone repository

2. Install dependencies

3. Jalankan development server

4. Buka di browser
